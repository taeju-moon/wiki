# 워드 클라우드를 활용한 트위터 감성 분석 시각화 실습

이번 과제는 트위터 데이터셋(`tweeter.csv`)을 활용하여 긍정/부정 텍스트 데이터를 전처리하고, `WordCloud` 라이브러리를 통해 주요 키워드를 한눈에 파악할 수 있도록 시각화하는 데이터 파이프라인 기초 실습 과제이다.

## 이번 프로젝트를 하며 배운 점들

**1. 정규표현식(Regex)을 활용한 텍스트 전처리 로직 구현**
URL, @멘션, 해시태그 기호(#), 특수기호 등을 제거하기 위해 `pandas.Series.str.replace()`와 정규표현식을 결합하여 노이즈 데이터를 깔끔하게 걷어내는 방법을 배웠다.

**2. 분석 목적에 맞는 커스텀 불용어(Stopwords) 정의**
트위터 특유의 은어('rt', 'u', 'lol')나 감성 분석에 큰 의미가 없는 일상적인 단어('today', 'going', 'think')들을 커스텀 불용어 사전에 직접 추가하여, 워드 클라우드의 품질과 분석의 정확도를 높이는 경험을 했다.

**3. `WordCloud` 라이브러리 객체 생성 및 활용**
전처리된 텍스트 데이터를 통째로 이어 붙인 뒤, `WordCloud()` 객체를 생성하여 최대 단어 수, 배경색, 컬러맵 등을 지정하고 `.generate()` 메서드로 손쉽게 단어 빈도수 기반 시각화 이미지를 만들어내는 방법을 익혔다.

**4. `matplotlib`을 이용한 긍정/부정 비교 시각화**
`plt.subplots(1, 2)`를 활용하여 긍정(Positive) 워드 클라우드와 부정(Negative) 워드 클라우드를 나란히 배치함으로써, 어떤 단어들이 각 감성을 주도하는지 직관적으로 비교 분석할 수 있는 능력을 길렀다.

## [추가 학습] 논문 요약 소감문
**논문:** *Predicting Election Results from Twitter Using Machine Learning Algorithms (Yadala et al., 2020)*

이 논문의 요약본을 읽으면서, 이번 과제에서 기초적으로 다루었던 '트위터 텍스트 전처리와 감성 분석'이 실제 현실 세계의 거대한 이벤트(선거)를 예측하는 데 얼마나 강력하게 응용될 수 있는지 깨달았다. 

특히 기존의 여론조사나 출구조사 같은 전통적인 통계 방식에 의존하지 않고, 대중이 실시간으로 쏟아내는 소셜 미디어 텍스트를 머신러닝(SVM)으로 분석하는 것만으로도 무려 94.2%의 높은 정확도로 선거 결과를 맞출 수 있다는 점이 매우 인상적이었다. 단순히 긍정과 부정 단어를 눈으로 확인하는 단계를 넘어서, 대중의 숨겨진 선호도를 파악하고 미래의 결과를 예측해내는 과정을 보며 텍스트 데이터와 머신러닝의 무궁무진한 잠재력을 느낄 수 있었다. 앞으로 정치 분야뿐만 아니라 기업이나 공공기관의 여론 모니터링 등 다양한 도메인에서 소셜 미디어 데이터 분석이 어떻게 활약할지 더욱 기대된다.
