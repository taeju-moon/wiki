# 파이썬 멀티프로세싱 병렬 처리 실습

이번 과제는 파이썬의 `multiprocessing` 모듈을 활용하여 여러 작업을 병렬로 동시에 처리하는 프로세스 풀(Pool) 스크립트를 구현하고 동작 원리를 이해하는 미니 실습 과제이다.

## 이번 프로젝트를 하며 배운 점들

**1. 프로세스 풀(Process Pool)을 활용한 자원 관리**
단순히 모든 프로세스를 한 번에 생성하는 것이 아니라, `multiprocessing.Pool(processes=2)`를 통해 워커(Worker) 프로세스의 개수를 2개로 제한하는 방법을 배웠다. 이를 통해 시스템 자원을 한꺼번에 소모하지 않고 효율적으로 관리하며 안정적으로 병렬 처리를 수행할 수 있음을 이해했다.

**2. `pool.map()`을 통한 작업 자동 분배**
리스트 형태로 된 여러 작업(A, B, C, D)과 소요 시간 데이터를 `pool.map()` 함수 하나로 간단하게 각각의 워커 프로세스에 할당하고 매핑하는 방법을 경험했다. 반복문을 직접 돌며 프로세스를 시작(start)하고 대기(join)하는 것보다 훨씬 간결하고 가독성 좋은 코드를 작성할 수 있었다.

**3. 병렬 처리 시나리오의 흐름 이해**
`time.sleep()` 함수를 사용하여 작업마다 다른 소요 시간(5초, 2초, 1초 등)을 모사해보았다. 풀에 할당된 2개의 프로세스가 동시에 동작하면서, 짧은 작업이 먼저 끝난 워커가 다음 대기열에 있는 작업을 곧바로 이어서 가져가 실행하는 일련의 흐름을 출력 로그를 통해 직관적으로 확인할 수 있었다. 이를 통해 순차적(Sequential) 실행 대비 전체 실행 시간을 효과적으로 단축하는 병렬(Parallel) 실행의 장점을 확실히 체감했다.
